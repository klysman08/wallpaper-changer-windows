# WallpaperChanger

Modos suportados:
- **random**  - imagem aleatoria de uma pasta
- **split2**  - tela dividida em 2 (esquerda/direita)
- **split4**  - tela dividida em 4 quadrantes

## Uso

```powershell
uv run wallpaper-changer apply
uv run wallpaper-changer apply --mode split2
uv run wallpaper-changer watch
```
